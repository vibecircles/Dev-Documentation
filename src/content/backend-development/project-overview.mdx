# Project Overview

The VibeCircles backend is a RESTful API that handles:
- User authentication and authorization
- User profile management
- Social features (follow/unfollow)
- Post creation and management
- Like/dislike functionality
- Timeline generation
- File uploads (images)

## Technology Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT (JSON Web Tokens)
- **File Upload:** Multer
- **Password Hashing:** bcrypt

VibeCircles is a social media web application backend built with the MERN stack (MongoDB, Express.js, React, Node.js). This documentation provides a comprehensive overview of the backend architecture, components, and API endpoints.
