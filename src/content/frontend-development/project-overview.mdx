# Project Overview

The VibeCircles frontend is a modern React-based social media application that provides:

- User authentication (Login/Signup)
- User profile management
- Social feed with posts
- Post creation and sharing
- Like and comment functionality
- Follow/Unfollow users
- Real-time updates

## Technology Stack

- **Framework:** React 18.2.0
- **State Management:** Redux with Redux Thunk
- **Routing:** React Router DOM v6
- **HTTP Client:** Axios
- **UI Library:** Material-UI (MUI) v5
- **Build Tool:** Create React App
- **Styling:** CSS Modules and CSS-in-JS

## Architecture Overview

VibeCircles frontend follows a component-based architecture with:

1. **Pages** - Top-level route components (Auth, Home, Profile)
2. **Components** - Reusable UI components organized by feature
3. **State Management** - Redux store with actions and reducers
4. **API Layer** - Centralized API request handlers
5. **Actions** - Redux action creators for async operations

## Key Features

- **Authentication Flow:** Secure login and registration with JWT tokens
- **Protected Routes:** Route guards based on authentication state
- **Real-time Updates:** Optimistic UI updates for better UX
- **Image Upload:** Support for profile pictures, cover images, and post images
- **Responsive Design:** Mobile-friendly interface with CSS Grid and Flexbox
